        margin apne parant ke hisab se pura contain move karta hai.
        (ex:- class=.x hai vo child hai or body parant hai.to div body ke hisab se move karega. )
         hame two box ke bich space chahiye to hum margin use karenge.

         margin: 100px 0 0 100px;
         (top right bottom left) 
         [yeh margin ki shorthand property hai(clockwise move karegi)]

         margin: 50px 50px ; 
         top/bottom =50px  right/Left=50px hogi

         margin: 50px;
         (direct margin 50px lene se clockwise (top,right,bottom,left) 50px apply hogi.)

         margin:auto;
         (left/right se center main hoga.)

        margin-top: 50px;
        (only top se 50px move hoga.)

        margin-right: 80px;
        (only right se 80px move hoga.)

        margin-bottom: 40px; 
        (only bottom se 40px move hoga.)

        margin-left: 100px
        (only left se 100px move hoga)

        text-aline: center 
        [center lese text ko center main(left/right se)  leke ayega.]

        line-hight:100px
        [line hight ko div ki hight dede to wo(top/bottom se) text ko center main kardega.]
