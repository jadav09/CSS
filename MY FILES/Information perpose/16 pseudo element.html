ager hame lot of h1 tag ke under koi  word or latter ko style dena hai.

==> note:- tag ko hum double colon(::) main hi likhenge. otherwise  not apply.


h1::first-letter :- h1  ke first latter ko style dega.
p::first-line    :- p tag ki first line ko style dega.

koi tag ke age-piche style through se likha na hai tab bhi uses hoga.

            content: "Hello ";
        }

            content: "Hello ";
        }

        }
