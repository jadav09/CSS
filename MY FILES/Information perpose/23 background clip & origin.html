      ==> The background-clip property defines how far the background should extend within an element.(background kitna dur rahega border se ye difind karega.)

      ==> background-clip:border-box (it is a default) property.(by default  background inside the border)

      ==> background-clip: padding-box: background padding area tak hi tahega.

      ==> background-clip: contant-box: background contant box tak hi tahega. 


      ==> background-origin concept == same as conditon for background:clip concept
